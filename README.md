# Dimplex Home Assistant Integration

A custom Home Assistant integration for Dimplex devices.

## Installation

### HACS (Recommended)

1. Open HACS in Home Assistant
2. Go to Integrations
3. Click the three dots menu and select "Custom repositories"
4. Add this repository URL
5. Install the integration

### Manual Installation

1. Copy the `custom_components/dimplex` folder to your Home Assistant `custom_components` directory
2. Restart Home Assistant
3. Go to Settings > Devices & Services > Add Integration
4. Search for "Dimplex" and follow the setup instructions

## Configuration

The integration can be configured via the Home Assistant UI:

1. Go to Settings > Devices & Services
2. Click "Add Integration"
3. Search for "Dimplex"
4. Enter your device details:
   - Host: IP address or hostname of your Dimplex device
   - Port: Port number (default: 80)
   - Name: Friendly name for the device

## Features

- Climate control
- Temperature monitoring
- HVAC mode control
